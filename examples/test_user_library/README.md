# Тестовая Пользовательская Библиотека

Это пример пользовательской библиотеки для Kumir 3, демонстрирующий:

- Конфигурацию через `kumir.toml`
- Определение функций на языке Kumir
- Экспорт функций для использования в других программах

## Структура

```
test_user_library/
├── kumir.toml          # Конфигурация библиотеки
├── src/
│   └── lib.kum         # Исходный код библиотеки
└── README.md           # Документация
```

## Функции

- `удвоить(цел x) -> цел` - удваивает число
- `утроить(цел x) -> цел` - утраивает число
- `сумма_чисел(цел a, цел b) -> цел` - возвращает сумму двух чисел
- `факториал(цел n) -> цел` - вычисляет факториал числа

## Использование

```kumir
использовать "путь/к/test_user_library"

алг Главный
нач
    вывод удвоить(5)        | 10
    вывод утроить(3)        | 9
    вывод сумма_чисел(2, 3) | 5
    вывод факториал(5)      | 120
кон
```
