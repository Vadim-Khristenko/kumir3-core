# KITE — стандарт языка Kumir 3

**KITE** (*Кумир: Инновации, Технологии, Эволюция*) — серия документов, описывающих язык
Kumir 3 (аналог Python PEP). Документы — это семантические HTML-фрагменты, встраиваемые в
страницу документации и оформленные в тёмной теме по умолчанию.

Открыть всё сразу: [`index.html`](index.html) — страница-оглавление с живым поиском.

## Как устроено

- Общий стиль и словарь классов `kite-*` — в [`assets/kite.css`](assets/kite.css) (тёмная тема по умолчанию).
- Прогрессивное улучшение — в [`assets/kite.js`](assets/kite.js): живой поиск по документу, генерация и подсветка оглавления (scrollspy), фильтр карточек на странице-оглавлении. Документы читаемы и без JavaScript.
- Каждый документ — фрагмент с корнем `<article class="kite">`; при автономном открытии красит всю страницу в тёмную тему. Подробности формата и процесса — в **KITE 0**.
- Базовый язык — русский; переводы (en) добавляются позже.

> Полное описание стандарта, статусов и процесса предложения новых KITE — в [KITE 0](KITE-0000-standard.html).
