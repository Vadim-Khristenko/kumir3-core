# Гайд на запуск

Чтобы запустить программу, используя интерпретатор, достаточно выполнить следующую команду

```bash
cargo run --bin interpreter-cli path_to_the_run_program 
```

или же, если вы хотите чуть более интересный вывод

```bash
cargo run --bin interpreter-tui path_to_the_run_program
```
