The legal files satisfy the lawyers, but the **KUL Experience** satisfies the **soul** of our codebase. By using or contributing to Kumir3-Core, you step into our "House Rules":

### 1. The Cat Protocol & "Hairball" Management 🐈

* **The Law:** If a cat interrupts your coding session, you are morally obligated to provide pets.
* **Technical Tie-in:** We treat messy, legacy code like "hairballs." If you find one, don't just step over it—refactor it. We strive for a codebase as clean and smooth as a cat's fur.

### 2. The High-Energy Standard (Fueling the Core) 🟢

* **The Ritual:** This core was built on **Monster Energy** (White/Ultra), **LemonFresh**, and high-quality hydration.
* **The Logic:** We write code that reflects this energy—fast, responsive, and efficient. We don't do "slow." If the code isn't snappy, it's not House-ready.

### 3. The Astolfo Aesthetic & BDoc Clarity ✨

* **The Philosophy:** Logic is only half the battle; clarity is the other half. We follow the **BetterDoc (BDoc)** standard.

* **The Mission:** Kumir is a Russian-native language. While our core comments are in English for global reach, we take pride in building the best possible tools for the Russian-speaking educational community. We bridge the gap between "accessible" and "high-tech."
